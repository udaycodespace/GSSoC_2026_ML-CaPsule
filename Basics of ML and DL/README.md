# Basics of ML and DL

This folder contains educational notebooks and examples covering core Machine Learning and Deep Learning topics.

Contents
- `ML/` - Classic ML tutorials: Linear/Logistic Regression, KNN, SVM, Random Forests, PCA, GridSearch, sample datasets and assignments.
- `Deep Learning/` - Introductory notebooks for Keras and TensorFlow.
- Subfolders with focused material: `NLP/`, `Pandas/`, `PCA/`, `GradientDescent_for_N-D_data/`, and more.

How to use
- Open the notebooks with Jupyter or VS Code and run cells sequentially.
- Recommended packages:

```bash
pip install pandas numpy scikit-learn matplotlib seaborn plotly tensorflow keras
```

Notes
- Each subfolder contains self-contained notebooks and small datasets. Start with `ML/LinearRegression_for_2-D_data.ipynb` or `Deep Learning/Keras Intro.ipynb` for quick introductions.
