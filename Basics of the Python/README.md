# Basics of the Python

Collection of Python learning materials and exercises grouped by topic.

Contents
- `Basic of Python/` — core language fundamentals (strings, lists, tuples, dictionaries, control flow, functions).
- `Data Visualizations/` — examples of plotting with Matplotlib and Plotly.
- `DSA with Python/` — data structures and algorithm implementations (trees, graphs, heaps, sorting).
- `ML Basics Models/` — small ML examples using scikit-learn and datasets.

How to use
- Open notebooks in the relevant subfolder to follow the lessons.
- Recommended packages:

```bash
pip install numpy pandas matplotlib seaborn scikit-learn
```

Learning path suggestion
- Start with `Basic of Python/` notebooks, then try `Data Visualizations/` and `ML Basics Models/` for applied examples.
