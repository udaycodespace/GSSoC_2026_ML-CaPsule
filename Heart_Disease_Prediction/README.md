# Heart Disease Prediction using Machine Learning

## 🎯 Goal
Predict whether a patient has heart disease using machine learning classification algorithms on real-world medical data.

## 📊 Dataset
- **Source:** UCI Heart Disease Dataset
- **Samples:** 303 patients
- **Features:** 13 medical attributes
- **Target:** 0 = No Disease, 1 = Heart Disease

## 📋 Features Description
| Feature | Description |
|---------|-------------|
| age | Age of patient |
| sex | Gender (0=Female, 1=Male) |
| cp | Chest pain type (0-3) |
| trestbps | Resting blood pressure |
| chol | Serum cholesterol |
| fbs | Fasting blood sugar |
| restecg | Resting ECG results |
| thalach | Maximum heart rate achieved |
| exang | Exercise induced angina |
| oldpeak | ST depression |
| slope | Slope of peak exercise ST |
| ca | Number of major vessels |
| thal | Thalassemia type |

- Chest pain type is the strongest predictor
- Max heart rate inversely correlates with disease
- Random Forest generally outperforms Logistic Regression
