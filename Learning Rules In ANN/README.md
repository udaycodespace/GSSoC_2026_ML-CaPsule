# Learning Rules In ANN

Notebooks and notes explaining learning rules used in artificial neural networks.

Contents
- `learning_rules.ipynb` — interactive notebook that demonstrates learning rules (Hebb, Delta rule, gradient descent variants).
- `Learning-Rules.md` — concise notes and formulas.

How to use
- Open the notebook to step through examples and visualizations of learning updates.
- Recommended packages:

```bash
pip install numpy matplotlib
```
